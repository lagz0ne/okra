# Worker Packet Action Envelope Global Only

Worker Prompt Packet Contract. Every next DKR or PKR dispatch is a packet.

- Frame fields: objective, anti_goals, metric_contracts, and frame_hash.
- Current state fields: current_state and current_round.
- Previous DKR checkpoint fields: source_dkr_checkpoint.
- Assignment fields: worker_kind and exact_task.
- Budget and stop rule, hand-back rule, and output schema.

Action envelope: allowed moves and forbidden moves are described later outside the frame fields.
