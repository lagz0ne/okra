# Candidate Anti-Goal Authority Drift Only

The accumulated anti-goal layer is a candidate guardrail library, represented as
`candidate-anti-goals.v1.json`. Each record includes `metric_id`, threshold, authority drift,
`applies_when`, `does_not_apply_when`, `invalidates_when`, source refs, `candidate_status`,
`no_regression_evidence`, and human ratification.

Previous-run anti-goals are candidate-only and not automatic authority.
