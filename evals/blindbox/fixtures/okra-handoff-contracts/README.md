# OKRA Handoff Contract Fixture

This fixture represents a repo where OKRA work repeatedly stalls at transition points. Prior runs
found these traps:

- a next worker prompt continued from a previous worker narrative instead of governed progress
  records
- a PKR was funded without a linked CKR contribution metric
- a PKR absorbed discovery by trying to figure out missing policy during implementation
- an accumulated anti-goal from a prior run was treated as active authority without human
  ratification

The desired artifact should define the prompt and handoff contracts needed to keep future runs from
repeating those failures.
